下面是一个将 DOCX 文件转换为 HTML 的 Python 脚本示例，使用了 python-docx 库来读取 DOCX 文件内容，和 html 库来生成 HTML 文件。

1. 安装必要的库

首先，确保你已经安装了 python-docx 和 html 库。如果还没有安装，可以通过以下命令安装：

pip install python-docx

2. Python 脚本
-------------------
from docx import Document
import html

def docx_to_html(docx_path, html_path):
    # 打开 DOCX 文件
    doc = Document(docx_path)

    # 初始化 HTML 内容
    html_content = "<html><body>"

    # 遍历每个段落，将其转换为 HTML 格式
    for para in doc.paragraphs:
        html_content += "<p>" + html.escape(para.text) + "</p>"

    # 遍历每个表格，将其转换为 HTML 格式
    for table in doc.tables:
        html_content += "<table border='1'>"
        for row in table.rows:
            html_content += "<tr>"
            for cell in row.cells:
                html_content += f"<td>{html.escape(cell.text)}</td>"
            html_content += "</tr>"
        html_content += "</table>"

    html_content += "</body></html>"

    # 将 HTML 内容写入文件
    with open(html_path, "w", encoding="utf-8") as html_file:
        html_file.write(html_content)

    print(f"文件已成功转换为 HTML: {html_path}")

# 使用示例
docx_path = "example.docx"  # 输入你的 DOCX 文件路径
html_path = "output.html"    # 输出的 HTML 文件路径
docx_to_html(docx_path, html_path)
=====================================
3. 说明

Document: python-docx 库用于打开 DOCX 文件。

html.escape(): 用于将文本中的特殊字符（如 <, >, & 等）转义为 HTML 实体，避免 HTML 标签错误。

段落转换: 将每个段落的文本转为 HTML 的 <p> 标签。

表格转换: 如果 DOCX 文件中有表格，将每个表格行和单元格转换为 <table>, <tr>, <td> 标签。

最后，生成的 HTML 内容会被写入指定路径的 .html 文件中。

4. 执行

运行这个脚本时，输入的 DOCX 文件将被转换为 HTML 文件，并保存到你指定的路径。
